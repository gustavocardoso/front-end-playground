Uma simples 'sandbox' para brincar com html, css e javascript!

Há algum tempo, venho fazendo vários pequenos experimentos para manter meus estudos com html, css e javascript. Depois de ter feito algumas "coisinhas legais", resolvi colocar tudo no Github e compartilhar com quem esteja interessado em aprender.

Aqui, brinco com coisas como css animation, css transitions, browser apis e afins. Juro que tento (ou tentarei) manter tudo sempre o mais atualizado possível. Ou não!

Vale lembrar que são apenas estudos e testes, ou seja, podem existir erros ou maneiras melhores de se fazer as mesmas coisas. Por isso, vou deixar aberto os comentários para receber os feedbacks construtivos.

Se você gostou, pode fazer um fork no Github ou mesmo contribuir com o projeto.
